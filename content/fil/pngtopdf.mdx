## Ano ang PNG?

### Kahulugan ng PNG
Ang PNG (Portable Network Graphics) ay isang lossless na naka-compress na format ng imahe. Ito ay unang inilabas noong 1996 at idinisenyo upang palitan ang GIF format.
### Mga katangian ng PNG

- Lossless compression: Ang mga PNG file ay hindi nawawala ang kalidad ng imahe dahil sa compression.
- Suporta sa Transparency: Sinusuportahan ng PNG ang mga transparent na background, na ginagawa itong napakapopular sa graphic na disenyo.
- Malawakang katugma: Ang format na PNG ay sinusuportahan ng halos lahat ng mga tumitingin at editor ng larawan.

### Mga karaniwang senaryo para sa paggamit ng PNG

- Web Design: Ang PNG ay malawakang ginagamit sa web design dahil sa transparency support nito.
- Pag-edit ng larawan: Ang walang pagkawalang compression ay ginagawa itong perpekto para sa pag-edit at pag-iimbak ng larawan.

## Ano ang PDF?
### 1. Kahulugan ng PDF
Ang PDF (Portable Document Format) ay isang format ng file na binuo ng Adobe Systems para kumatawan sa mga dokumento, kabilang ang pag-format ng teksto at mga imahe.
### 2. Mga Tampok ng PDF
- Cross-platform compatible: Maaaring tingnan at i-print ang PDF sa anumang operating system.
- Mataas na katapatan: Ang mga PDF file ay patuloy na ipinapakita sa iba't ibang device.
- Multifunctional: sumusuporta sa teksto, mga imahe, mga link, mga form at iba pang mga elemento.

### 3. Mga karaniwang senaryo para sa paggamit ng PDF
- Ang PDF ay napaka-maginhawa kapag nagbabahagi ng mga dokumento sa pagitan ng iba't ibang device at operating system.
- Ang mga PDF file ay nagpapanatili ng kanilang orihinal na layout at pag-format kapag naka-print.
- Ang PDF ay ang gustong format para sa pag-archive ng mga dokumento dahil sa katatagan nito at pangmatagalang pagiging madaling mabasa.

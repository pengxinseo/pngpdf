## Apa itu png

png (Portable Network Graphics) adalah format file gambar yang banyak digunakan dan dikenal karena dukungannya terhadap kompresi dan transparansi lossless. Kompresi lossless berarti file png tidak kehilangan kualitas gambar setelah diedit dan disimpan beberapa kali, sehingga ideal untuk menyimpan gambar yang perlu menjaga kualitas tinggi, seperti ikon dan desain grafis. File PNG mendukung latar belakang transparan, membuatnya banyak digunakan dalam desain web, terutama dalam skenario di mana gambar dan latar belakang perlu dipadukan secara mulus.

## Apa itu pdf

PDF (Portable Document Format) adalah format file yang digunakan untuk mewakili dokumen. Baik pada sistem operasi yang berbeda atau perangkat yang berbeda, file PDF dapat mempertahankan format dan tata letak aslinya. Format PDF mendukung berbagai tipe konten seperti teks, gambar, tabel, link, dan multimedia, menjadikannya format pilihan untuk dokumen elektronik. File PDF juga memiliki tingkat keamanan yang tinggi, dan perlindungan kata sandi serta kontrol izin dapat diatur untuk memastikan kerahasiaan dan integritas konten dokumen. Oleh karena itu, file PDF banyak digunakan untuk penerbitan dan berbagi kontrak, laporan, e-book, dan dokumen formal lainnya.
